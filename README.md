## Installation
1. Navigate to the backend project directory
```bash
cd prj300ExpressServer2
```

2. Install dependencies
```bash
npm install
```

### Environment Variables

Create a .env file in the root directory and add the following:

PORT=3000
DB_CONN_STRING= YOUR_STRING
DB_NAME= YOUR_NAME

Running the Server
Development mode
```bash
npm run dev
```



### The API will run at:

http://localhost:3000
